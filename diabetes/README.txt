##############################
# Descrição da Base de Dados #
##############################


1. idade - A idade varia de 0 a 80 em nosso conjunto de dados.

2. Género - Gênero refere-se ao sexo biológico do indivíduo. 
	feminino = 0, masculino = 1, e outros = 2.

3. Hipertensão - A hipertensão é uma condição médica na qual a pressão sanguínea nas artérias é persistentemente elevada. 
	0 = indica que não tem hipertensão.
	1 = significa que tem hipertensão.

4. doença cardíaca - Existência de doença cardíaca. 
	0 = indica que não tem doença cardíaca 
	1 = significa que tem doença cardíaca.

5. historicoFumante - O histórico de tabagismo. 
	0 = nunca fumou
	1 = fumante
	2 = ex-fumante
	3 = não informado pelo paciente

6. IMC - IMC (Índice de Massa Corporal) é uma medida de gordura corporal com base no peso e na altura. 

7. nivelAcucar - O nível de HbA1c (Hemoglobina A1c) é uma medida do nível médio de açúcar no sangue de uma pessoa nos últimos 2-3 meses. 

8. glicose - O nível de glicose no sangue.

9. diabetes - presença de diabetes.  
	1 - indicando a presença de diabetes 
	0 - indicando a ausência de diabetes.
